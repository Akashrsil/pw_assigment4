Diffrence Between HTML and HTML5

HTML HTML5
The original version of HTML, which has evolved over time with various versions such as HTML 4.01 and XHTML. The fifth major revision of HTML, introducing new features, elements, and APIs aimed at improving the functionality, performance, and accessibility of web content.
Did not include a built-in mechanism for drawing graphics or animations directly in the browser. Introduces the canvas element, which provides a JavaScript-based drawing API for rendering 2D graphics, animations, games, and other visual content directly within the browser, without the need for plugins or external libraries.
Includes a limited set of semantic elements such as header, footer, and nav, but these were not standardized across all browsers. Introduces several new semantic elements such as header, footer, nav, article, section, and aside, providing better structure and meaning to web documents and improving accessibility and search engine optimization (SEO).
